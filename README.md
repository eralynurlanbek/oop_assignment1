## Project Description
Simple console-based library application in Java.  
You can add, search, borrow, return, and delete books.

## Classes
- `Rectangle` – represents a rectangle with width, height, and ID
- `Book` – represents a book with title, author, year, availability, and ID
- `LibraryApp` – console application with menu and user interaction
